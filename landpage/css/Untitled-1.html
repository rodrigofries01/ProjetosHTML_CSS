2021-1 NETWORK ESSENCIAL REDES DE COMPUTADORES Iniciado em quarta, 5 mai 2021, 18:35 Estado Finalizada Concluída em quarta, 5 mai 2021, 19:13 Tempo empregado 38 minutos 37 segundos Notas 26,00/26,00 Avaliar 100,00 de um máximo de 100,00 Questão 1 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Que termo se refere ao conjunto de regras que definem como uma rede funciona? Escolha uma opção: padrão protocolo modelo domínio Feedback Refer to curriculum topic: 3.1.2 Os padrões garantem que dispositivos diferentes conectados à rede consigam se comunicar entre si. Em redes, há muitos padrões que definem que todos os componentes implementem os mesmos protocolos e como a rede deve operar. A resposta correta é: padrão. Questão 2 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Qual é a finalidade da camada do núcleo no modelo de projeto de rede hierárquico da Cisco? Escolha uma opção: acesso à rede para dispositivos finais comutação de backbone em alta velocidade ponto de agregação para redes menores controle de fluxo entre redes Feedback Refer to curriculum topic: 3.4.1 Na hierarquia de três camadas, a camada de acesso fornece conectividade de host à rede. A camada de distribuição interconecta as redes menores, e a camada do núcleo fornece conexões de alta velocidade no topo da hierarquia. A resposta correta é: comutação de backbone em alta velocidade. Questão 3 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Que tipo de modelo de rede descreve as funções que devem ser completadas em uma camada específica, mas não especifica exatamente como cada protocolo deve trabalhar? Escolha uma opção: Modelo TCP/IP modelo de protocolo modelo de referência modelo do design hierárquico Feedback Refer to curriculum topic: 3.1.3 Também há dois tipos comuns de modelos de comunicação de rede:  Modelo de protocolo - esse módulo corresponde intimamente à estrutura de um conjunto de protocolos específico e descreve as funções que ocorrem em cada camada de protocolos dentro do conjunto. O modelo TCP/IP é um exemplo de modelo de protocolo.  Modelo de referência - o objetivo principal é auxiliar em um entendimento mais claro das funções e dos processos necessários para a comunicação de rede. Esse tipo de modelo não especifica exatamente como uma função deve ser realizada. O modelo OSI é um exemplo de modelo de referência. A resposta correta é: modelo de referência. Questão 4 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Que tipo de informação contém uma tabela ARP? destino de mensagem tipo de mensagem origem da mensagem dados de mensagem meio de transmissão comum prioridade de mensagem Feedback Refer to curriculum topic: 3.1.1 Todos os métodos de comunicação incluem uma origem de mensagem, destino e um meio de transmissão. As respostas corretas são: destino de mensagem, origem da mensagem, meio de transmissão comum Questão 9 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão O que possibilita que e-mails sejam enviados e recebidos em vários tipos de dispositivos, inclusive celulares, PDAs, notebooks e computadores desktop? Escolha uma opção: Todos os dispositivos executam o mesmo software de sistema operacional. Há um único fornecedor de software de servidor de e-mail. Os dispositivos usam o mesmo software de cliente de e-mail e por isso são compatíveis. O software de e-mail é gravado usando padrões e protocolos que ajudam a garantir a compatibilidade. Feedback Refer to curriculum topic: 3.1.2 A resposta correta é: O software de e-mail é gravado usando padrões e protocolos que ajudam a garantir a compatibilidade.. Questão 10 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Que termo é usado para se referir a uma PDU de camada 2? Escolha uma opção: pacote quadro segmento bit Feedback Refer to curriculum topic: 3.3.1 A unidade de dados de protocolo (PDU) da camada 2 é o quadro. A resposta correta é: quadro. Questão 11 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Corresponda os componentes na notação 100Base-T à especificação. (Nem todas as opções são usadas). 100 Resposta 1 BASE Resposta 2 T Resposta 3 Feedback Refer to curriculum topic: 3.2.2 De acordo com o padrão 802.3, os componentes na notação 100Base-T indicam:  100 é a velocidade em Mbps  BASE significa transmissão de banda base  T significa o tipo de cabo, neste caso, par trançado. A resposta correta é: 100 → velocidade em Mbps, BASE → transmissão banda base, T → cabo de par trançado. Questão 12 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Quais são as duas camadas do modelo OSI que especificam os protocolos associados aos padrões Ethernet? (Escolha duas.) Escolha uma ou mais: camada física camada de enlace de dados camada de rede camada de transporte camada de sessão Feedback Refer to curriculum topic: 3.2.2 Os padrões Ethernet especificam os protocolos que operam na camada 1 (camada física) e 2 (camada de link de dados) do modelo OSI e é a tecnologia usada por quase todas as redes da área local conectadas. As respostas corretas são: camada física, camada de enlace de dados Questão 13 Correto Atingiu 2,00 de 2,00 Marcar questão Texto da questão Quando um computador monta um quadro a ser enviado pela rede, qual é o tamanho máximo de um quadro Ethernet? Escolha uma opção: 64 bytes 128 bytes 1024 bytes 1518 bytes Feedback Refer to curriculum topic: 3.3.1 Os padrões Ethernet definem o tamanho de um quadro com um mínimo de 64 bytes e um máximo de 1518 bytes que incluem campos MAC de destino, MAC de origem, comprimento/tipo, payload de dados e de FCS. A resposta correta é: 1518 bytes.
